# batman
this is batman symbol made by the help of python language and turtle graphics """Batman is a superhero who appears in American comic books published by DC Comics. Batman was created by artist Bob Kane and writer Bill Finger, and debuted in the 27th issue of the comic book Detective Comics on March 30, 1939."""


## Installation

Use the package manager [pip](https://pypi.org/project/turtle/) to install foobar.

```bash
pip install turtle
```

## Usage

```python
import turtle
```
